How to Use

Musafir helps you avoid adult content and set focus sessions with a timer. Basic steps:

  • Click the Musafir icon to open the popup and select a timer (for example, 25 minutes or 1 hour).
  • Blocked sites will be redirected to an informative blocked page.
  • Media flagged as inappropriate is blurred automatically on pages you visit.
  • View timer status and controls in the popup; stop or extend sessions as needed.

Troubleshooting

Timer won't reset

Reload Musafir at chrome://extensions/ or toggle the extension off/on. If the issue persists, restart your browser.

Blocking not working

Ensure Musafir is enabled and has the correct permissions. Open the extension console with Ctrl+Shift+J (Windows) to check for errors.

Popup fails to open

Clear browser cache and try again. If needed, reinstall the extension from your store or the provided CRX.

Media blurring not applied

Some sites use dynamic content or heavy obfuscation. Report the page via the contact form so we can improve detection rules.

FAQ

Can I whitelist sites?

Whitelist functionality is planned. For now, use the extension options to manage exceptions where available.

Is my data collected?

Musafir does not collect personal browsing data. Any reports you submit are used to improve detection and support only.
